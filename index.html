Setting a background colour for a section header

The above header is set to green. Here I use colour coding to help you find the section corresponding to the bulleted table of contents. A class can be assigned a style. I focused on distinguishing background colours for all of my list items in the table of contents, and their corresponding <h2> headings throughout this cheat sheet. For example, I created a class called "sections", which I added as an attribute for the list item and heading. Two ways to do this depending on the desired goal are described below

Approach Tag/Attribute Format Explanation
list li with class attribute <li class="sections"> This approach highlights the whole width of the screen
mark mark with class attribute <li><mark class="sections"> This approach highlights only the text. Mark defaults to yellow as a highlight colour.

See this page to learn about mark.

In the css sheet, I added a style to all elements with the "sections" class. This over-rode the default color of yellow for <mark> elements.

  .sections {
   background-color: lawngreen;
  }
