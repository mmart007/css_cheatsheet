Setting a background colour for a section header

The above header is set to green. Here I use colour coding to help you find the section corresponding to the bulleted table of contents. A class can be assigned a style. I focused on distinguishing background colours for all of my list items in the table of contents, and their corresponding <h2> headings throughout this cheat sheet. For example, I created a class called "sections", which I added as an attribute for the list item and heading. Two ways to do this depending on the desired goal are described below

Approach Tag/Attribute Format Explanation
tag with class tag [li/h2] with class attribute <h2 class="sections"> This approach highlights the whole width of the screen
mark as embedded tag mark with class attribute <h2><mark class="sections"> This approach highlights only the text. Mark defaults to yellow as a highlight colour.

See this page to learn about mark.

In the css sheet, I added a style to all elements with the "sections" class. This over-rode the default color of yellow for <mark> elements. I will repeat for each of my 4 subjects in the bullet list, each of which has its own class.

  .sections {
   background-color: lawngreen;
  }

Emphasizing text in a paragraph

If I want to emphasize text, I can format a subset of the text using the <span> tag and assign a class. In my case, I have used my class called emphasis that uses the <mark> tag with a color property of red.
